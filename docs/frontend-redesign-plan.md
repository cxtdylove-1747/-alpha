# 创新创业智能体平台前端重设计方案（基于现有前后端）

## 1. 设计目标

在不改变现有后端接口与业务流程的前提下，升级页面视觉质感与交互细节：

- 风格：简约高级 + 科技感 + 轻商务
- 原则：可维护、可落地、低风险改造
- 范围：先落地学生端 `智能引导` 页面，作为统一视觉样式基准

## 2. 视觉系统

### 2.1 色彩

- 主色：`#146eff`
- 成功：`#4cd964`
- 提醒：`#ffd76b`
- 风险：`#ff7875`
- 辅助强调：`#9d9dff`
- 标题：`#1a1a2e`
- 正文：`#3a3a48`
- 辅助文字：`#8a8a9a`
- 页面背景：`linear-gradient(180deg, #f8f9fc 0%, #f0f2f5 100%)`
- 边框：`#ebecef`

### 2.2 字体层级

- 页面主标题：20px / 700
- 模块标题：18px / 700
- 卡片标题：16px / 600
- 正文：14px / 400
- 辅助：12px / 400

### 2.3 组件

- 卡片：圆角 10px，阴影 `0 2px 10px rgba(0,0,0,0.03)`
- 主按钮：渐变 `#146eff -> #0f5edb`，圆角 8px
- 输入框：圆角 8px，聚焦边框主色 + 2px 发光
- 标签：圆角 6px，低饱和背景

## 3. 页面信息架构（本次落地页）

页面：学生端 `智能引导`（`/student/home`）

- 左栏（功能辅助区）
  - 学习路径（时间轴）
  - 引导摘要（当前阶段、问题数）
  - 推进趋势图（ECharts 折线）
- 右栏（核心工作区）
  - 智能对话主面板（历史会话 + 输入 + 发送/校验）

## 4. 交互与状态

- 发送问题：调用 `/api/ai/chat`
- 逻辑校验：调用 `/api/ai/validate`
- 历史会话：本地存储 `studentChatHistory`
- 新建会话：归档当前并清空对话上下文

## 5. 开发映射

- 样式基线：`frontend/src/assets/css/global.css`
- 顶栏导航：`frontend/src/components/NavBar.vue`
- 页面主体：`frontend/src/pages/student/HomeGuide.vue`
- 对话组件：`frontend/src/components/AgentChatWindow.vue`

## 6. 验证清单

- 页面可正常打开、路由可切换
- 提问与逻辑校验接口返回可渲染
- 新建会话与历史切换正常
- 在 1366 和 1920 宽度下布局无错位
